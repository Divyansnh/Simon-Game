How to Play Advanced Simon

1. Press Start to begin the game.

2. Watch and remember the sequence of colors.

3. Repeat the sequence by clicking the colored buttons.

4. The sequence gets longer with each round.

5. If you make a mistake, the game ends.

Game Modes:

- Classic: Standard Simon game

- Speed: Faster sequence playback

- Reverse: Repeat the sequence in reverse order
